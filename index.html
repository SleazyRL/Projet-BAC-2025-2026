Prototype

La manette sur
breadboard.

Avant de concevoir le PCB final, un prototype complet de la manette a été réalisé sur breadboard. Chaque bouton est connecté directement aux broches GPIO du Raspberry Pi — les entrées numériques gèrent chaque touche individuellement. Les joysticks sont gérés par un Arduino Micro en USB.

  • Broches GPIO Raspberry Pi — entrées numériques
  • Arduino Micro USB — gestion des joysticks analogiques
  • Extension 2×20 pins GPIO incluse
  • Breadboard + fils — 13,99€ de prototype

Mapping

Chaque bouton
configuré.

Au premier démarrage de RetroPie, chaque touche de la manette doit être mappée une par une. D-PAD, boutons ABXY, bumpers, gâchettes — chaque entrée GPIO est associée à une touche clavier reconnue par l'émulateur.

  • D-PAD — 4 directions mappées sur Z, Q, S, D
  • Boutons ABXY — touches H, G, T, F
  • Bumpers + gâchettes — U, E, X, C
  • Démarrer → Entrée · Sélectionner → Maj
Alimentation

8 à 10h
d'autonomie.

Le Raspberry Pi 4B nécessite une alimentation USB-C 5V / 3A. La PiSugar 2Pro de 5000 mAh répond exactement à ce besoin — elle se connecte via les broches pogo à l'arrière du Raspberry Pi, offrant jusqu'à 8–10h de jeu non-stop sans fil ni connecteur supplémentaire.

  • PiSugar 2Pro — 5000 mAh
  • Autonomie — 8 à 10h de jeu continu
  • Connexion pogo — à l'arrière du RPi
  • 5V / 3A — alimentation USB-C compatible

RetroPie

Tous les rétro
en un seul OS.

RetroPie est l'OS choisi pour sa communauté massive et sa facilité d'utilisation. Il regroupe des dizaines d'émulateurs — NES, SNES, Game Boy, PlayStation 1 — accessibles depuis une interface unifiée. L'installation se fait en flashant l'image sur la carte SD puis en l'insérant dans le Raspberry Pi.

  • Basé sur Linux — Raspberry Pi OS
  • NES · SNES · Game Boy · PSX
  • Installation via carte SD 128Go
  • EmulationStation — interface unifiée
Écran

Écran 5 pouces
800 × 480.

L'écran choisi est un 5 pouces en format 5:3 à 800×480 pixels — résolution parfaitement adaptée aux jeux rétro. Il se connecte via micro HDMI (avec adaptateur plat 90°) et s'alimente en USB. La configuration se fait dans les paramètres d'affichage de RetroPie.

  • 5 pouces — format 5:3 — 800×480px
  • Connexion micro HDMI vers HDMI
  • Adaptateur plat 90° — gain de place
  • Résolution optimale pour l'émulation rétro
