# Projet-BAC-2025-2026
FR:
J’ai voulu faire cette console car je suis fan de jeux retro de type gameboy , nes , playstation 1 , 2 et 3. Grace a l’émulateur Retro Pie disponible sur Raspberry pi je pourrais réunir tous mes jeux retro préférés sur une seule console portable.Pour réaliser ce projet je me suis fixé un budget max de 350€ . 

EN: 
I wanted to build this console because I’m a fan of retro games like Game Boy, NES, PlayStation 1, 2 and 3. Thanks to the RetroPie emulator available on the Raspberry Pi, I’ll be able to bring all my favourite retro games together on a single portable console. To carry out this project, I’ve set myself a maximum budget of 350€
